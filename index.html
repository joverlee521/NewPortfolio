What is Left or Write?

Left or Write is a cloud-based "Choose Your Own Adventure" reading and writing platform.

The main goal of the project is to create a user friendly platform that does not require writers to learn special syntax to write branching fiction. Other key features of this application are it's mobile friendly and it allows writers to create unfinished story lines so they can write episodically.

Major Technologies Used:

  • Bootstrap
  • Handlebars
  • Node
  • Express
  • Sequelize
  • Passport.js

What is Full Keto?

Full Keto is a single page application that is dedicated to the Ketogenic lifestyle.

The main goal of the project is to design a simple and elegant UI that allows users to search for Keto recipes and nutrition information of specific food items.

Major Technologies Used:

  • React
  • Material-UI
  • Node and Express
  • Passport
  • EDAMAM API
  • Nutritionix API

What is Entertain Me?

Entertain Me is a single page application for searching events.

The main goal of the project is to create a straight-forward UI that allows users to search for events based on location, date, and event category. The search results contains details about the event and a Google map that allows users to map to the event location.

Major Technologies Used:

  • Materialize
  • JavaScript
  • Eventful API
  • Google Maps API
  • Algolia Places
